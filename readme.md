# Portfolio v4

This repository contains the fourth version of my personal portfolio website. It showcases selected UI/UX, product design, and frontend projects in both English and Traditional Chinese, and includes a playful home page with custom animations and a "Kitchen" section.

## Highlights

- Bilingual portfolio experience: English and Traditional Chinese
- Project detail pages for design and frontend work
- Custom homepage interactions and character animation
- Reusable data-driven content rendering
- SCSS-based styling workflow
